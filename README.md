# AST_xingkong
你看到的是一个AST处理的压缩包,默认我把依赖上传了,但不代表你本身没有这个软件,就可以使用
1.本软件主要用处:可以辅助你将乱七八糟的混淆进行还原,我们原理是通过前期AST剔除平坦,以及处理了变量,生成新的代码,然后减少体积,交给大模型处理,最后还原处理
这个时候就用人问了,为什么不直接叫给AI处理,而是做这么多前置需求呢?原因很简单,混淆有几万行,甚至有几十万行都可能正常,因此不能破坏结构,另外也会影响AI分析.进行前期处理
2.本软件禁止用于违法用途,仅用于学习，禁止用于商业开发。未经作者允许,禁止私自商业行为。
3.默认是支持V6 V7 OB safekodo 这些混淆 但是这里声明,开发本软件,仅仅用于学习,禁止用于破坏别人网站,窃取机密,若您用于该软件,窃取机密,请您自行负责。开发者只分享开发劳动成功。
